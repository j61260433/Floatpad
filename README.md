# 🏦 Floatpad - Overdraft Protection Protocol

> 💰 Backup funds with dynamic interest rates for seamless liquidity management

## 🌟 Overview

Floatpad is a decentralized overdraft protection protocol built on Stacks that provides users with backup liquidity when they need it most. Think of it as a safety net for your finances - deposit funds to earn yield, or borrow against collateral with dynamic interest rates.

## ✨ Key Features

- 🏊‍♂️ **Liquidity Pools**: Create and manage liquidity pools with custom interest rates
- 💎 **Collateralized Borrowing**: Borrow up to 150% of your collateral value
- 📈 **Dynamic Interest Rates**: Rates adjust based on pool utilization
- ⚡ **Instant Overdraft**: Get immediate access to funds when you need them
- 🛡️ **Liquidation Protection**: Automated liquidation system to protect lenders
- 💰 **Yield Generation**: Earn interest on deposited liquidity

## 🚀 Getting Started

### Prerequisites

- [Clarinet](https://github.com/hirosystems/clarinet) installed
- Stacks wallet with STX tokens

### Installation

```bash
git clone <your-repo>
cd floatpad
clarinet check
```
